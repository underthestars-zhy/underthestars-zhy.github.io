What is python

Wiki: Python（英国发音：/ˈpaɪθən/ 美国发音：/ˈpaɪθɑːn/）是一种广泛使用的解释型、高级编程、通用型编程语言，由吉多·范罗苏姆创造，第一版发布于1991年。Python是ABC语言的后继者，也可以视之为一种使用传统中缀表达式（英语：M-expression）的LISP方言。Python的设计哲学强调代码的可读性和简洁的语法（尤其是使用空格缩进划分代码块，而非使用大括号或者关键词）。相比于C+ +或Java，Python让开发者能够用更少的代码表达想法。不管是小型还是大型程序，该语言都试图让程序的结构清晰明了。

简单来说，python就是一款简单的编程语言，底端语言是C，所以运行比较快速。
目前，python已经被加入中学生计算机课程，所以学好它还是比较有用的。

How to use Python

安装

  1. MacOS，前往https://python.org/downloads下载，系统自带3.6版本，和2.X版本
  2. Windows，前往https://python.org/downloads下载
  3. Linux，一般系统自带，可以尝试运行相关命令检测

运行

安装以后，会得到一个IDLE的软件。打开它。

>>>print("Hello Python")

输入以上内容，回车，会得到

Hello Python

在这个上面，>>>是运行符号，代码都在这个后面运行。

写

在上方找到New File，新建，输入：

print("First File")

然后保存，再按下F5（Windows）运行
页面会跳转到IDLE，出现：

First File
