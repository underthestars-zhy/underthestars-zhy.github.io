Typecho从0开始

Typecho从0开始

本次要介绍的是从0开始搭建一个属于自己的Typecho程序。

安装宝塔

安全组

我们首先需要放行服务器安全组，直接一键放行所有出入安全组，即可。

如果你的服务器不能配置安全组就不能使用

宝塔安装

在远程链接输入以下代码

yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh

如果不是Centos无法使用，Centos请装7.3

然后会出现安装界面，请等待一会（看你服务器速度），当出现以下代码则是安装完成

Bt-Panel: http://xx.xxx.xx.xx:8888/xxxxxx
username: xxxxx
password: xxxxxx

到这里宝塔安装完成了。

环境配置

然后，打开Bt-Pane这个链接，这个就是你的宝塔，输入用户名与密码登陆。会出现安装，选择第一个安装选项卡（Nginx），在这里宝塔会帮你一键将环境配置好。有个Php版本选择7.3。

Typecho安装

搭建

首先在网站这里选择新建

输入域名，数据库选择Mysql，Php选择7.3。

点击保存，会跳出一个Mysql账户密码界面，把这个保存以下

然后点击网站名称，在里面找到ssl，选择Letxxxx这个选项，选择文件验证，并且打勾你的域名，签名SSl

注意把域名用A记录解析过来

SSL配置好以后选择强制https

安装

点击根目录，然后下载此文件

下载

上传到刚刚打开的目录，然后解压，这个时候会出现一个B开头的文件，打开这个文件，然后全选，剪贴到上一个目录。

数据库配置

访问你绑定域名会出现配置界面。将刚刚复制的Mysql的用户名粘贴到数据库名称和用户名这两个选项，密码就粘贴到密码。再输入你的管理员用户名与密码。

然后，OK了

插件，主题

主题

把你要安装的主题上传到/usr/themes，然后解压。到typecho后台的外观启用

插件

把你要安装的主题上传到/usr/plugins，然后解压。到typecho后台的插件启用
