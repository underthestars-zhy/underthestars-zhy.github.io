Typecho文章自定义字体CSS

CSS代码

在后台——外观设置——自定义CSS中填入以下代码

@font-face{
			font-family: 'FZZ';
			src:url('https://cdn.jsdelivr.net/gh/s-cloud-bk/Upic@master/uPic/字魂162号-元气酪酪体.ttf');
		}

.box{
	font-family: 'FZZ';
	font-size: 30px;
}

其中，src部分的url可以换为你想要的字体，FZZ可以替换，但是保证上下名称一样，box也可以换，等会调用的时候需要一起替换
